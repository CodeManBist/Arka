This directory contains skill definitions for AI agents working on the Blast Radius project.

## 🎯 Skill Categories

### Analysis Skills
- **Code Analysis**: Parsing, AST generation, symbol extraction
- **Graph Analysis**: Call graph, import graph, traversal algorithms
- **Impact Analysis**: Risk scoring, confidence calculation, diff parsing

### Development Skills
- **Frontend**: Next.js, React, TypeScript, Tailwind CSS
- **Backend**: Node.js, Express, API design
- **AI Service**: FastAPI, Python, Tree-sitter integration

### Testing Skills
- **Unit Testing**: pytest, Jest, test coverage
- **Integration Testing**: End-to-end testing, API testing

### Deployment Skills
- **Docker**: Containerization, multi-service deployment
- **Cloud**: AWS, GCP, Azure deployment strategies

## 🤖 Usage

AI agents should reference these skill files when working on specific tasks. Each skill file contains:
- Required knowledge and prerequisites
- Common patterns and best practices
- Examples and code snippets
- Troubleshooting tips
- Related resources

## 📝 Adding New Skills

To add a new skill:
1. Create a new markdown file in the appropriate category
2. Follow the template structure
3. Include practical examples and code snippets
4. Link to related documentation
5. Update this README with the new skill
